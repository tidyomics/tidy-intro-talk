Tidyomics project

An open, open-source project spanning multiple R packages, and developers from around the world. Operates within the Bioconductor Project, with separate funding and organization. Coordination of development via GitHub Projects.

Keep data organized in objects

  • We typically have more information than just a matrix

  • Row and column information (on features and samples)

  • Annotated matrix data, i.e. python’s xarray, AnnData

  • Metadata: organism, genome build, annotation release

  • Endomorphic functions: x |> f -> x

Computing on genomic ranges with tidy grammar

Nontrivial to translate biological or biomedical questions about the genome into code. The tidy grammar helps in this case with familiar concepts of filters, joins, grouping, mutation, or summarization.

nullranges for defining null genomic range sets

We developed a package nullranges, as a modular tool to assist with making genomic comparisons. It doesn’t do enrichment analysis but provides null genomic range sets for investigating various hypotheses.

Outro

Omics data analysis means looking at:

  • main contributions to variance (e.g. PCA, see plotPCA for bulk and OSCA for sc), also variancePartition
  • column and row densities (tidySE allows directly plotting geom_density of rows/columns, or geom_violin)
  • known positive features, feature-level plots (filter to feature, pipe to geom_point etc.)

If you’re interested in more complicated use cases of tidyomics see this online book:
